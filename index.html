jQuery-inlog

jQuery-inlog allows you to easily debug your jQuery code. See exactly what you pass to functions, what they return and in which context (this-value) they where executed.

  • No more alert($obj.length) to see if your selector worked.
  • No more pain when chaining many calls and figuring out which one fails

NEW: See nested or recursive calls in the right order. Wooot.

Without changing any of your code! I call it black-box magic. Or short: black magic. Or even shorter: js

Documentation / Reference

Include jquery.inlog.js after jQuery core and you're done. See options if you want more control.

Options

You can call $.inlog(options) any time to change options. Available options:

var defaults = {
	enabled: true, //Shortcut: $.inlog(true|false);
	thisValue: false, //Output this-value or not
	returnValue: true, //Output return-value or not
	indent: false //Indent nested calls or not
};
